AuditoriumX - Theater Booking System
====================================

This is a portable desktop application for theater booking management.

HOW TO USE:
1. Double-click "AuditoriumX.exe" to start the application
2. The application will automatically start the backend server
3. The web interface will open in the application window

SYSTEM REQUIREMENTS:
- Windows 10 or later
- No additional software installation required
- All dependencies are included

FILES INCLUDED:
- AuditoriumX.exe (Main application)
- resources/ (Application resources and backend)
- All necessary libraries and dependencies

DISTRIBUTION:
- Copy the entire "app" folder to any Windows computer
- The application is completely self-contained
- No installation required

TROUBLESHOOTING:
- If the application doesn't start, try running as Administrator
- Make sure no other application is using port 3001
- Check Windows Defender/Firewall settings if needed

For support, contact the development team.
